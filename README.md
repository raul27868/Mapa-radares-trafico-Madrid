# Mapa de radares (Madrid)

Mapa estático para GitHub Pages que lee el XLSX de radares del Ayuntamiento de Madrid y pinta marcadores por Longitud/Latitud mostrando la “Velocidad límite”.

## Publicar en GitHub Pages
1. Sube estos ficheros al repo.
2. En **Settings → Pages**:
   - Source: `Deploy from a branch`
   - Branch: `main` / folder: `/ (root)`
3. Abre la URL de Pages.

## Datos
Proceden del Portal de Datos Abiertos del Ayuntamiento de Madrid (dataset de radares).
